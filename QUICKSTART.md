# ⚡ Quick Start

## Installation

```bash
git clone <repo-url>
cd email-intent-classification
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Train Model

```bash
python src/train_model.py
```

## Run GUI

```bash
python src/gui_app.py
```

## Use Predictor

```bash
python src/predict_engine.py
```
